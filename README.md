# 📍 Consulta de CEP - Java Spring Boot

API REST desenvolvida em Java para consumo de serviços externos, permitindo a busca automatizada de endereços através do CEP.

## 🚀 Tecnologias
* **Java 17**
* **Spring Boot 3**
* **Spring Cloud OpenFeign** (Consumo de API externa)
* **Maven** (Gerenciamento de dependências)

## 🛠️ Funcionalidades
* Integração em tempo real com a API **ViaCEP**.
* Conversão automática de JSON para objetos Java (DTOs).
* Endpoint customizado para consulta rápida.

## 🏁 Como rodar
1. Clone o repositório.
2. Certifique-se de ter o Maven instalado.
3. Execute o comando: `mvn spring-boot:run`.
4. Acesse: `http://localhost:8080/cep/01001000`.
